Basic TODO application with create, update, delete features.

Installation Details
1. to go to the cmd, click on down arrow in terminal of vscode, click select default profile, select command prompt or cmd option)
2. kill the terminal and press ctrl + z to open in cmd mode
3. create virtual environment (python -m venv venv)
4. activate venv (ctrl + shift + p, select interpreter, click on venv recommended), if doesnot appear restart vscode
5. kill the terminal and press ctrl + z, now the venv env will be activated
6. pip install django
7. create Django project (django-admin startproject TodoApplication .)
8. create app (python manage.py startapp myApp)
9. Run server (python manage.py runserver)
10. python manage.py migrate
11. python manage.py runserver
12. ctrl + click to the port address to open to a new browser
13. go to /admin to go to admin page
14. python manage.py createsuperuser
15. after creating a model use the following commands(python manage.py makemigrations, python manage.py migrate)
16. pip freeze to see the requirements
